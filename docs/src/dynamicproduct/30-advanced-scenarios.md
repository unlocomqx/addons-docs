# Advanced pricing

## Using a grid for pricing

If your pricing depends on two fields, then you can use a pricing grid to easily configure it.

1. Add a width, height and cost fields
2. Configure your grid like the screen capture below
3. Add your cost field to the [pricing formula](/dynamicproduct/product-config/08-formulas.md)

<img src="/dynamicproduct/images/grid-example@2x.png" alt="Grid example" width="798" height="337">

You can also use a [CSV file](/dynamicproduct/15-formula-functions.md#the-grid-function) to configure your pricing. This way, you can use the same pricing for multiple products.
