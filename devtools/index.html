# Introduction

DevTools

DevTools is a PrestaShop module which provides a set of tools to help you edit the theme templates and hooks easily.

With this module, you can edit templates directly in the browser and modify the hooks without switching to the backoffice.

When you enable the module, you will see the hooks and template blocks clearly, which allows you to see the theme structure and edit it easily.
